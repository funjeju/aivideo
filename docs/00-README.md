## 핵심 원칙 5가지 (모든 파일에 공통 적용)

1. **저마찰 (Low-friction)**: 사용자는 최소한의 선택(주제/문서 + 3개 옵션)만 한다. 시안 고르기 단계는 없다.
2. **콘텐츠 동결, 형식 변환**: '충실 모드'에서 LLM은 사실·논리·용어를 바꾸지 않고, 형식(구어체·길이·장면분할)만 변환한다.
3. **결정적 렌더링**: 같은 장면 JSON → 항상 같은 영상. 편집은 JSON만 수정 후 부분 재렌더.
4. **화풍 = 교체 가능한 Style Pack**: 정보 구조(텍스트·도형 문법)는 고정, 그림체만 팩으로 교체.
5. **비용 통제**: 영상 1편의 외부 API 원가를 항상 추적 가능하게 설계 (`04-pipeline.md`의 cost 로깅 참조).

## ⚠️ Claude Code가 사용자에게 반드시 먼저 요청해야 할 것

구현 시작 전, 아래 항목은 **코드에 하드코딩하지 말고 사용자에게 요청**한다 (`05-design-system.md`, `02-architecture.md`에 상세):

- [ ] **shadcn/ui MCP 연결** — 컴포넌트 설치/조회를 MCP로 진행. 미연결 시 사용자에게 연결 요청.
- [ ] Firebase 프로젝트 자격증명 (`.env`용)
- [ ] OpenAI API 키 (GPT Image 2 + TTS)
- [ ] LLM API 키 (원고/연출 생성용 — Claude 또는 GPT)
- [ ] Google Cloud 프로젝트 (Cloud Run 배포용)
- [ ] 액센트 컬러 최종 확정 (`05-design-system.md` 기본값: 단청 적색)

## 용어 빠른 참조

- **Style Pack**: 그림체 1종 = (이미지 프롬프트 템플릿 + 텍스처 오버레이 + 연출 파라미터).
- **장면 JSON / Scene Spec**: 영상의 단일 진실 소스(single source of truth). 렌더러는 이것만 보고 영상을 만든다.
- **렌더링 워커 (Render Worker)**: 오래 걸리는 영상 굽기 전용 서버 (Cloud Run). Vercel 밖에 있음.
- **생성 모드 / 충실 모드**: 입력 방식 2종. 상세는 `04-pipeline.md`.
- **role**: 사용자 권한 (`user`/`staff`/`superadmin`). 어드민 영역은 `07-admin.md`.
- **uiLocale / contentLocale**: UI 언어 / 영상 콘텐츠 언어. 두 레이어 분리 (`05-design-system.md` 6.6).
